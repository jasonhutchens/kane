KANE
====

Share your best link each day.

Design
------

* User
  + name
  + profile
  + settings
  +
  + shares
  + stats
  + curators

* Profile
  + image
  + name

* Settings
  + subscribed
  + credits
  + timezone

* Share
  + position
  + user
  + content

* Content
  + title
  + description
  + body
  + publish date
  + url
  + creator
  + tags
  + thumbnail
  + type (banned, external, naive, premium)
  + language
  +
  + shares
  + stats

* Creator
  + name
  + favicon
  + url
  + contents

* Stat
  + user
  + content
  + reaction
  + saved / deleted / whatever
  + seen count
  + view count
  + total time
  + percentage viewed

* Reaction
  + emoji
  + stats
